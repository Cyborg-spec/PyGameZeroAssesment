# PyGameZeroAssesment
## How to Run

Install Pygame Zero:
pip install pgzero

Make sure folders "images" and "sounds" are in the same directory as main.py.

Run the game using:
pgzrun main.py

## Libraries Used

math (built-in)

random (built-in)

pgzrun (Pygame Zero)

pygame.Rect

## Game Description
Dungeon Runner is a simple top-down game. The player moves around the dungeon, avoids enemies, and tries to reach the exit. Each level increases difficulty with faster enemies, more enemies, and added interior walls.

## Controls

Arrow Keys / WASD: Move

Mouse: Click menu buttons and continue after win/lose

## Features

Main menu with Start, Sound Toggle, Exit

Idle + walking animations for player and enemies

Two enemy types with different sprites

Collision system using custom colliders

Increasing difficulty each level

Interior walls from Level 2 onward

Background music and sound effects

Win when reaching exit, lose on enemy collision
